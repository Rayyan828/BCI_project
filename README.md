## Project Overview

This project implements and extends the research paper:

**Title:** Prediction of Cognitive Load from Electroencephalography Signals Using Long Short-Term Memory Network  
**Journal:** Bioengineering (MDPI)  
**Link:** [https://www.mdpi.com/2306-5354/10/3/361](https://www.mdpi.com/2306-5354/10/3/361)  

The goal is to predict cognitive load from EEG signals using deep learning models, specifically LSTM networks, and compare the performance with additional models.

**Dataset Used:** SAM-40 Cognitive Load EEG Dataset
